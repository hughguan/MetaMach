# 🛡️ MetaMach

> **Act on the Meta Layer.** Stop loop engineering. Build the system that builds the system.

MetaMach is a high-performance, ultra-lightweight, and 100% deterministic **AI Software Factory OS**. It orchestrates specialized AI agents (Claude Code, Codex, Pi) as isolated, ephemeral function nodes inside robust engineering pipelines—all powered by remote hardware and managed straight from your pocket.

---

### 🏗️ Architecture Matrix

*   **The Muscle (`Herdr`):** Manages non-volatile, isolated agent sandboxes inside remote PTY pane layouts via local socket APIs.
*   **The Nervous System (`Hermes`):** Pure bash/python automation control. Zero-token overhead routing, conditional flows, and lint/test netting.
*   **The Observatory (`Obsidian`):** Live terminal dumping to Markdown. Turning telemetry into a compounding local knowledge base.
*   **The Tentacle (`Telegram`):** Stateless, zero-latency mobile proxy for high-dimensional, inline-button human-in-the-loop validation.

---

### 📱 The MetaMach Workflow Experience

1.  **Tactical Intervention:** Pocket vibrates. Telegram drops a precise diff/error card. Click `[🔄 Refactor]` or `[⚡ Merge]` right on your phone.
2.  **Zero-Friction Observation:** Open your laptop, type `metamach`, and instantly teleport to the pure, un-nested remote TUI matrix with zero local fan noise or CPU drain.
